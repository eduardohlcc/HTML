Iate Club — Aluguel de Barcos

Site institucional de uma empresa fictícia de aluguel de iates de luxo, desenvolvido com HTML e CSS puros.

Sobre o Projeto:

O Iate Club é um projeto web estático criado para apresentar uma empresa de aluguel de embarcações de luxo. O site conta com seções informativas sobre destinos e frota, além de um layout elegante e responsivo.
Este projeto foi desenvolvido com fins educacionais, praticando estruturação de páginas com HTML semântico e estilização com CSS.

Este projeto está sob uso livre para fins educacionais.
